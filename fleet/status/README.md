# Fleet Status Files

This folder is the GitHub-visible fleet dashboard.

## Open First

- `current.md`: latest status, overwritten every report cycle.
- `today.md`: hourly progress log for the current day.
- `archive/`: prior daily logs.

The top section of `current.md` is the captain summary. It is meant to answer:

- Is the project running, ready, parked, blocked, or stalled?
- What changed recently?
- What is the next task?
- Is the branch clean and synced?

For the full guide, see `FLEET_REPORTS_README.md` at the repo root.
