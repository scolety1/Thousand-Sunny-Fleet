# Manifest Producer Authority

The durable-result mapper resolves strong evidence through the verified compact manifest. Each required record must have one expected logical type, fixed relative path, producer, allowed classification, recorded full SHA-256 hash, and independently recomputed matching hash inside the packet.

Mission, revision, run/result, policy, repository, branch, worktree, queue-document, thread, turn, event, usage, kernel, verifier, and preservation relationships are then revalidated. Caller paths must exactly equal the manifest-selected packet paths; structural consistency alone grants no evidence classification.

The prompt is preserved as `q.txt` with `KERNEL_OBSERVED` provenance. Adapter stderr is preserved as `se.log` with `UNVERIFIED` diagnostic classification and cannot become strong evidence.
