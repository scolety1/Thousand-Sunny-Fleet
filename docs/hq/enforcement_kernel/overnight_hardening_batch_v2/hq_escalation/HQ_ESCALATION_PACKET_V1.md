# HQ Escalation Packet V1

This packet shape prepares TSF for future ChatGPT/API HQ choke-point review without implementing API access in V2.

It carries:

- mission summary
- preflight result
- verifier result
- blocker register
- changed files summary
- evidence snippets
- exact decision requested
- allowed verdicts
- required JSON response shape

No API call is made by this schema. A future API/HQ integration still requires exact Tim approval, redaction rules, transport rules, and response-validation rules.
