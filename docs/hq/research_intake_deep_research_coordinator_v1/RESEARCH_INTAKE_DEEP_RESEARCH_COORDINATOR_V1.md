# Research Intake / Deep Research Coordinator V1

Verdict: GREEN_RESEARCH_PIPELINE_LOCAL_COORDINATOR_READY

This packet defines the local-only research pipeline. It normalizes ideas, plans bounded prompt packages, exports manual Deep Research packets, imports returned reports, and synthesizes advisory recommendations. It does not submit research, call APIs, access credentials, run background workers, mutate product repos, or inspect canonical NWR.

## Flow

Idea capture -> duplicate and depth check -> bounded prompt plan -> manual export package -> manual external research by Tim if approved -> import returned report -> preserve immutable report and metadata -> synthesize advisory decision -> Tim/HQ adoption gate.

## Authority Boundary

Research evidence can recommend KEEP / CHANGE / ADD / REMOVE / DELAY. It cannot approve implementation, push, merge, API use, background runners, product repo mutation, canonical NWR mutation, source-truth changes, ranking changes, or app wiring.
