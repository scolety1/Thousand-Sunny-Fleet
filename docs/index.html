Rules

Phone edits are requests, not execution authority. Remote access does not grant extra authority.
  • Safe: read status, capture a narrow idea, submit a bounded quick mission request.
  • Use laptop or desktop for Codex, terminal checks, local validation, commits, pushes, or code changes.
  • Do not approve deploys, installs, migrations, secrets, all-fleet, overnight runners, lock deletion, permission widening, or runtime command binding from phone text.
