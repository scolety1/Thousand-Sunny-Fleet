## Notes

This is the captain's "are we actually ready?" sheet.

## Implemented Scorecard

The readiness scorer accepts checks with:

```text
stage
scenario
category
status
evidence
openRisk
requiredFix
```

Statuses:

```text
PASS
PASS_WITH_FIXES
FAIL
```

Verdict rules:

| Condition | Verdict |
| --- | --- |
| all required stages/scenarios covered and all PASS | READY_FOR_CONTROLLED_USE |
| coverage complete and at least one PASS_WITH_FIXES | READY_WITH_LIMITS |
| missing coverage or any FAIL | NOT_READY |

Implemented command:

```text
invoke-final-readiness.ps1 -InputPath <checks.json>
invoke-final-readiness.ps1 -UseExampleFixture
```

Example scorecard output is generated by the focused Stage 14 tests.
